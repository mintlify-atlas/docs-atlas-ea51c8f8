## Overview

Flash STU is an optimized, open-source PyTorch implementation of the Spectral Transform Unit (STU) as proposed in [*Spectral State Space Models*](https://arxiv.org/abs/2312.06837) by Agarwal et al. (2024). This implementation complements the [Flash STU: Fast Spectral Transform Units](https://arxiv.org/abs/2409.10489) paper.

The STU module is a fast and flexible building block that can be adapted into a wide range of neural network architectures, especially those that aim to solve tasks with long-range dependencies.

## What is STU?

The Spectral Transform Unit (STU) is a neural network module that leverages spectral methods to efficiently capture long-range dependencies in sequence data. By operating in the frequency domain, STU can process sequences with significantly better computational efficiency compared to traditional attention mechanisms, making it ideal for:

- Language modeling with long contexts
- Time series analysis
- Signal processing tasks
- Any application requiring efficient long-range dependency modeling
