<Warning>
  CUDA is required to run Flash STU. Ensure you have a CUDA-compatible GPU and drivers installed.
</Warning>

## System Requirements

Flash STU has been tested with the following configuration:

- **Python**: 3.12.5
- **PyTorch**: 2.4.1
- **Triton**: 3.0.0
- **CUDA**: 12.4

<Note>
  Flash STU may be incompatible with other versions. If you encounter issues, please ensure you're using the tested versions above.
</Note>

## Installation Steps

<Steps>
  <Step title="Install PyTorch with CUDA support">
    Install PyTorch 2.4.1 with CUDA 12.4 support:

    ```bash
    pip install torch --index-url https://download.pytorch.org/whl/cu124
    ```

    This installs PyTorch compiled with CUDA 12.4, which is required for GPU acceleration.
  </Step>

  <Step title="Install Flash STU dependencies">
    Install the required packages for Flash STU:

    ```bash
    pip install -e .
    ```

    This will install all dependencies specified in the `setup.py` or `pyproject.toml` file.
  </Step>

  <Step title="Install Flash Attention">
    Install Flash Attention for fast local attention operations:

    ```bash
    MAX_JOBS=4 pip install flash-attn --no-build-isolation
    ```

    <Note>
      The `MAX_JOBS=4` environment variable limits parallel compilation jobs. Adjust this based on your system's available memory and CPU cores.
    </Note>

    <Note>
      The `--no-build-isolation` flag is required for proper compilation of Flash Attention.
    </Note>
  </Step>

  <Step title="Install Flash FFT">
    Install Flash FFT for fast convolution operations:

    ```bash
    pip install git+https://github.com/HazyResearch/flash-fft-conv.git#subdirectory=csrc/flashfftconv
    pip install git+https://github.com/HazyResearch/flash-fft-conv.git
    ```

    These commands install both the core Flash FFT library and its CUDA kernels.
  </Step>
</Steps>

## Alternative Installation

You can also install Flash STU directly from the GitHub repository:

```bash
pip install git+https://github.com/windsornguyen/flash-stu.git
```

This will install Flash STU and its Python dependencies, but you'll still need to install Flash Attention and Flash FFT separately following Steps 3 and 4 above.

## Verify Installation

To verify your installation, run the following Python code:

```python
import torch
from flash_stu import FlashSTU, FlashSTUConfig, get_spectral_filters

print(f"PyTorch version: {torch.__version__}")
print(f"CUDA available: {torch.cuda.is_available()}")
print(f"CUDA version: {torch.version.cuda}")

if torch.cuda.is_available():
    print(f"GPU: {torch.cuda.get_device_name(0)}")
    print("Flash STU is ready to use!")
else:
    print("WARNING: CUDA is not available. Flash STU requires CUDA.")
```

## Troubleshooting

### CUDA Not Available

If CUDA is not available, ensure:
- You have a CUDA-compatible NVIDIA GPU
- NVIDIA drivers are properly installed
- The CUDA toolkit version matches your PyTorch installation

### Flash Attention Build Errors

If you encounter build errors with Flash Attention:
- Reduce `MAX_JOBS` to avoid out-of-memory errors during compilation
- Ensure you have sufficient disk space for compilation artifacts
- Check that your CUDA toolkit is properly configured

### Flash FFT Import Errors

If Flash FFT fails to import:
- Verify both Flash FFT packages are installed correctly
- Check that your PyTorch and CUDA versions are compatible
- The code will automatically fall back to PyTorch FFT if Flash FFT is unavailable
